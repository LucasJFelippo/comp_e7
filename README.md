# Compiladores - Entrega 7

 Entrega da atividade 7 de Compiladores, aonde foi implmentado modulos simples para C com operações básicas.

## Operações Implementadas

- Multiplicação (multi.c);
- Divisão Inteira (div.c);
- Módulo (mod.c);
- Potencia (power.c).

## Instruções

Para compilar basta digitar:

```
make
```

Para rodar basta digitar:

```
make run
```

Para limpar basta digitar:

```
make clean
```
